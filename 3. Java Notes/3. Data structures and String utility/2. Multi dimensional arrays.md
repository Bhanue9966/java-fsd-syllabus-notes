## 2\. Multi-Dimensional Arrays

A **Multi-Dimensional Array** is essentially an **array of arrays**. The most common type is the **Two-Dimensional (2D) Array**, which can be visualized as a grid or a table.

### 1\. Two-Dimensional (2D) Arrays

A 2D array represents data in a matrix format, defined by **rows** and **columns**.

#### A. Declaration and Initialization

A 2D array requires two sets of square brackets (`[][]`)‚Äîone for the row index and one for the column index.

**1. Declaration:**

```java
dataType[][] arrayName; 
// OR
dataType arrayName[][];
```

*Example:*

```java
int[][] matrix;
```

**2. Initialization (Fixed Size):**
You must specify the number of rows and the number of columns.

```java
// 3 rows and 4 columns
int[][] matrix = new int[3][4]; 
```

**3. Initialization (Literal):**
You can initialize the array directly using a nested initializer list.

```java
int[][] grid = {
    {10, 20, 30}, // Row 0
    {40, 50, 60}  // Row 1
}; 
// This creates a 2x3 matrix
```

### 2\. Accessing Elements

Elements in a 2D array are accessed using **two indices**: `[row index][column index]`.

```java
int[][] grid = {
    {1, 2, 3}, 
    {4, 5, 6}  
};

// Accessing the element at Row 1, Column 0 (the number 4)
int value = grid[1][0]; 
System.out.println(value); // Output: 4

// Modifying an element
grid[0][2] = 99; // Changes 3 to 99
```

### 3\. Iterating 2D Arrays (Nested Loops)

Since a 2D array is an array of arrays, you need **nested `for` loops** to iterate over all elements:

  * **Outer Loop:** Controls the **rows** (`i`).
  * **Inner Loop:** Controls the **columns** (`j`) within the current row.

<!-- end list -->

```java
int[][] data = {{1, 2}, {3, 4}};

for (int i = 0; i < data.length; i++) { // Loop through rows
    // data[i].length gives the size of the current row (number of columns)
    for (int j = 0; j < data[i].length; j++) { // Loop through columns
        System.out.print(data[i][j] + " ");
    }
    System.out.println(); // Move to the next line after a row is printed
}
// Output:
// 1 2 
// 3 4 
```

### 4\. Jagged Arrays (Irregular Arrays)

A **Jagged Array** is a multi-dimensional array where the **length of each inner array (row) can be different**. This is possible because Java treats a 2D array as an array of references to 1D arrays.

  * **Creation:** You must specify the number of rows, but leave the column dimension empty. Then, you initialize each row individually.

<!-- end list -->

```java
// 1. Declare and initialize 3 rows (but columns are undefined)
int[][] jaggedArray = new int[3][]; 

// 2. Initialize each row with different sizes
jaggedArray[0] = new int[2]; // Row 0 has 2 columns
jaggedArray[1] = new int[4]; // Row 1 has 4 columns
jaggedArray[2] = new int[1]; // Row 2 has 1 column

// 3. Assignment (Example)
jaggedArray[1][3] = 77; 
```
