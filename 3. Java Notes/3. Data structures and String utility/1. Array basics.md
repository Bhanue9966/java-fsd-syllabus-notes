## 1\. Arrays Basics (Declaration and Initialization)

An **Array** is a linear, fixed-size data structure that holds a sequence of elements of the **same data type**. It is the most basic data structure in Java.

### 1\. Key Characteristics of Arrays

  * **Fixed Size:** Once an array is created, its size (number of elements) **cannot be changed**.
  * **Homogeneous:** It can only store elements of the **same data type** (e.g., an array of `int` can only hold integers).
  * **Contiguous Memory:** Array elements are stored in adjacent (contiguous) memory locations. This allows for very fast, direct access to any element using its index.
  * **Object:** Arrays in Java are treated as **objects**, created on the **Heap** memory.

### 2\. Array Declaration and Initialization

There are several ways to declare and initialize an array in Java.

#### A. Declaration (Defining the type)

The declaration tells the compiler the variable is an array reference. It does not allocate memory for the elements.

```java
dataType[] arrayName; 
// OR (less common but valid): 
dataType arrayName[]; 
```

*Example:*

```java
int[] numbers; 
String[] names; 
```

#### B. Initialization (Allocating Memory)

Memory is allocated using the **`new`** keyword, where you must specify the **size** of the array.

```java
// Allocates memory for 5 integers
int[] numbers = new int[5]; 
```

  * **Default Values:** When initialized, array elements are automatically assigned default values based on their type:
      * **Numeric types** (`int`, `double`, etc.): `0` or `0.0`
      * **`boolean`:** `false`
      * **Reference types** (`String`, etc.): `null`

#### C. Declaration + Initialization + Assignment (In One Line)

You can declare, allocate, and assign values in a single statement using an array initializer list:

```java
// The size is automatically determined by the number of values (3 elements)
String[] names = {"Alice", "Bob", "Charlie"};
```

### 3\. Accessing and Modifying Elements

Array elements are accessed using a zero-based **index**.

  * **Index Range:** The index starts at `0` and goes up to `size - 1`.
  * **Accessing:** `arrayName[index]`

<!-- end list -->

```java
int[] data = {10, 20, 30};

// Accessing:
int firstElement = data[0]; // 10

// Modifying:
data[1] = 25; // Change the second element from 20 to 25

// Accessing the length:
int arrayLength = data.length; // 3
```

> ‚ö†Ô∏è **`ArrayIndexOutOfBoundsException`:** Trying to access an element using an index outside the valid range (e.g., `data[3]` or `data[-1]` in the example above) will cause a runtime exception.

### 4\. Array Length Property

Every array object has a **`length`** field that stores the number of elements in the array. It is a `final` field, meaning the size cannot be changed after creation.

```java
int[] grades = new int[50];
int size = grades.length; // size will be 50
```
