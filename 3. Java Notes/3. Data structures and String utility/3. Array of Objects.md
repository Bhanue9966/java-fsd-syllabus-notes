## 3\. Array of Objects

An **Array of Objects** is an array whose elements are **references** to objects of a particular class. It is the bridge between primitive arrays and more flexible data structures like Collections.

### 1\. Declaration and Creation

Creating an array of objects is a two-step process:

1.  **Declare and Create the Array Container:** This allocates memory for the array itself, which will hold references. All element references are initialized to **`null`**.
2.  **Create the Actual Objects:** You must iterate through the array and create a new object for each element, assigning the object's reference to the array index.

#### Example: `Student` Class

First, define the class we want to store:

```java
public class Student {
    String name;
    int rollNo;

    public Student(String name, int rollNo) {
        this.name = name;
        this.rollNo = rollNo;
    }

    public void display() {
        System.out.println("Name: " + name + ", Roll No: " + rollNo);
    }
}
```

#### Steps in Creating an Array of `Student` Objects

| Step | Description | Code Example |
| :--- | :--- | :--- |
| **1. Declaration/Allocation** | Create an array variable and allocate space for 3 object references. All 3 slots currently hold `null`. | `Student[] students = new Student[3];` |
| **2. Object Instantiation** | Use the constructor (`new Student(...)`) to create the actual objects and assign their references to the array indices. | `students[0] = new Student("Alice", 101);` |

### 2\. Complete Code Example

```java
public class ArrayOfObjectsDemo {
    public static void main(String[] args) {
        // Step 1: Create the array container (contains 3 null references)
        Student[] students = new Student[3];
        
        // Step 2: Create and assign objects to the array indices
        students[0] = new Student("Alice", 101);
        students[1] = new Student("Bob", 102);
        students[2] = new Student("Charlie", 103);
        
        // Step 3: Accessing and using the objects
        System.out.println("--- Array Elements ---");
        for (int i = 0; i < students.length; i++) {
            // Call a method on the object referenced at index i
            students[i].display(); 
        }
    }
}
// Output:
// --- Array Elements ---
// Name: Alice, Roll No: 101
// Name: Bob, Roll No: 102
// Name: Charlie, Roll No: 103
```

### 3\. Accessing and Common Errors

  * **Accessing a Member:** You access the object first using the index, then access its member using the dot operator (`.`).
    ```java
    // Accessing the 'name' field of the student at index 1
    String bobName = students[1].name; 
    ```
  * **The Danger of Null:** If you try to access a member before instantiating the object, you will get a **`NullPointerException`**.
    ```java
    Student[] students = new Student[1]; // Index 0 is null
    // students[0].display(); // RUNTIME ERROR: NullPointerException
    ```

### 4\. Direct Initialization (Shorthand)

You can combine steps 1 and 2 if you know all the objects you want to create upfront:

```java
Student[] students = {
    new Student("David", 104),
    new Student("Eve", 105)
};
// This array has a length of 2 and contains two fully initialized objects.
```
