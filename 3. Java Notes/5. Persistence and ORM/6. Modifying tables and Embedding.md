## 6. Hibernate: Naming Conventions and Embeddables

This section covers how to customize the mapping between Java and the database when default naming conventions don't match, and how to structure complex value objects using the `@Embedded` and `@Embeddable` annotations.

### 1. Custom Naming Conventions üè∑Ô∏è

By default, Hibernate maps class names to table names and field names to column names. When your database uses different conventions (e.g., `snake_case` for columns), you use specific annotations to override this default mapping.

| Annotation | Location | Purpose |
| :--- | :--- | :--- |
| **`@Table(name="...")`** | On the Entity Class | Maps the Java class to a specific database table name. |
| **`@Column(name="...")`** | On the Entity Field | Maps the Java field to a specific database column name. |

### 2. Embeddable Objects (Value Types) üéÅ

An **embeddable object** allows you to structure a complex field within your Java entity (like an `Address` or `Name`) while keeping all its fields in the **same table** as the main entity.

* **Embeddable Class (Value Object):** The class being embedded must be annotated with **`@Embeddable`**. It typically has no primary key (`@Id`).
* **Entity Class (Owner):** The entity that contains the value object must use the **`@Embedded`** annotation on the field.



### Example: Using Embeddable for `FullName`

We will modify the `Student` entity to remove `sName` and replace it with an embeddable `FullName` object containing `firstName` and `lastName`.

#### 2. Update the Entity Class (`Student.java`)

The `Student` class is updated to use the embeddable object and custom table/column names.


```java
package com.telusko;

import jakarta.persistence.Embeddable;

// @Embeddable marks this class as a value type that can be included 
// directly into an entity's table structure.
@Embeddable
public class FullName {
    
    // Custom column names are used here to demonstrate the @Column annotation
    // firstName -> f_name column
    // lastName -> l_name column
    
    // Note: No @Id is present here, as it is not a primary entity
    private String firstName;
    private String lastName;

    public FullName() {
    }

    public FullName(String firstName, String lastName) {
        this.firstName = firstName;
        this.lastName = lastName;
    }

    // Getters and Setters
    public String getFirstName() {
        return firstName;
    }

    public void setFirstName(String firstName) {
        this.firstName = firstName;
    }

    public String getLastName() {
        return lastName;
    }

    public void setLastName(String lastName) {
        this.lastName = lastName;
    }

    @Override
    public String toString() {
        return firstName + " " + lastName;
    }
}
```

#### 3. Main Demonstration Class (`NamingAndEmbedDemo.java`)

The main class demonstrates how to create and persist a `Student` object that contains an instance of the `FullName` embeddable class.


```java
package com.telusko;

import org.hibernate.Session;
import org.hibernate.SessionFactory;
import org.hibernate.Transaction;
import org.hibernate.cfg.Configuration;

/**
 * Demonstrates the use of @Embedded/@Embeddable and custom naming 
 * (@Table, @Column) in Hibernate.
 */
public class NamingAndEmbedDemo {
    public static void main(String[] args) {

        SessionFactory sf = new Configuration()
                .configure()
                .addAnnotatedClass(Student.class)
                .addAnnotatedClass(FullName.class) // Add the embeddable class as well
                .buildSessionFactory();

        final int DEMO_ROLL_NO = 111;

        try (Session session = sf.openSession()) {
            Transaction transaction = null;

            // ==================================================================
            // A. CREATE (PERSIST) with Embedded Object
            // ==================================================================
            try {
                transaction = session.beginTransaction();
                
                // 1. Create the Embeddable Value Object
                FullName name = new FullName("Virat", "Kohli");
                
                // 2. Create the Entity, assign the Value Object
                Student s = new Student();
                s.setRollNo(DEMO_ROLL_NO);
                s.setStudentName(name);
                s.setsAge(35);
                
                session.persist(s);
                transaction.commit();
                System.out.println("‚úÖ CREATE: Student " + s.getRollNo() + " (with Embeddable Name) persisted.");
            } catch (Exception e) {
                if (transaction != null) transaction.rollback();
                System.err.println("‚ùå CREATE failed: " + e.getMessage());
            }

            // ==================================================================
            // B. READ Operation and Displaying the Embedded Object
            // ==================================================================
            System.out.println("\n--- B. READ ---");
            Student retrievedStudent = session.find(Student.class, DEMO_ROLL_NO);
            
            if (retrievedStudent != null) {
                System.out.println("‚úÖ Retrieved Student: " + retrievedStudent);
                System.out.println("   Full Name (from Embeddable): " + retrievedStudent.getStudentName().toString());
                
                // Note: The database table 'student_details' now has columns 
                // 'student_roll', 'student_age', 'firstName', and 'lastName'.
                
            } else {
                System.out.println("‚ùå Student " + DEMO_ROLL_NO + " not found.");
            }

        } finally {
            if (sf != null) {
                sf.close();
                System.out.println("\nSession and SessionFactory closed.");
            }
        }
    }
}
```
