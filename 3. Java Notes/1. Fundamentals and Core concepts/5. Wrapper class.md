## 5\. Wrapper Class

### üì¶ What is a Wrapper Class?

A **Wrapper Class** is a class in the `java.lang` package whose object literally **wraps or contains** a primitive data type.

In Java, we deal with two types: **primitives** (like `int`, `boolean`) and **objects** (like `String`). Primitives are fast but lack object capabilities. Wrapper classes provide the necessary object counterpart for every primitive.

| Primitive Type | Corresponding Wrapper Class |
| :--- | :--- |
| `boolean` | `Boolean` |
| `char` | `Character` |
| `byte` | `Byte` |
| `short` | `Short` |
| `int` | `Integer` |
| `long` | `Long` |
| `float` | `Float` |
| `double` | `Double` |

### ‚ùì Why Do We Need Wrapper Classes?

1.  **Collections Framework:** Java's **Collection Framework** (e.g., `ArrayList`, `HashMap`) can only store **objects**, not primitives. You must use `Integer` instead of `int`.
2.  **Null Value:** Primitive types cannot be assigned `null`. Wrapper objects can, which is useful when dealing with optional values or database fields.
3.  **Utility Methods:** Wrapper classes provide helpful utility methods (e.g., converting a string to a number).

### üîÑ Autoboxing and Unboxing (Java 5+)

Java automatically handles the conversion between primitives and wrappers through two features:

#### 1\. Autoboxing

The **automatic conversion** of a **primitive type** into its corresponding **Wrapper Class object**.

```java
int primitiveInt = 50;

// Autoboxing happens here: int is converted to an Integer object
Integer wrapperInt = primitiveInt; 
```

#### 2\. Unboxing

The **automatic conversion** of a **Wrapper Class object** back into its corresponding **primitive type**.

```java
Integer wrapperInt = 100;

// Unboxing happens here: Integer object is converted back to an int primitive
int primitiveInt = wrapperInt; 
```

### üí° Useful Conversion Methods

Wrapper classes offer static methods to manually convert types, especially strings:

| Method | Wrapper Class | Description | Example |
| :--- | :--- | :--- | :--- |
| **`parseXxx()`** | `Integer.parseInt()` | Converts a **String** to a **primitive** type. | `int i = Integer.parseInt("123");` |
| **`valueOf()`** | `Integer.valueOf()` | Converts a **String** or **primitive** to a **Wrapper object**. | `Integer w = Integer.valueOf("456");` |
