## 10\. For Loop and Enhanced For Loop

The **`for` loop** is the most widely used loop in Java, primarily because its structure allows the initialization, condition check, and update steps to be defined cleanly on a single line.

### 1\. The Standard `for` Loop

The standard `for` loop is ideal when you know the **exact number of iterations** you need to perform.

#### Syntax

The control components are separated by semicolons (`;`):

```java
for (initialization; condition; update) {
    // Code block to be executed repeatedly
}
```

| Component | Execution | Example |
| :--- | :--- | :--- |
| **Initialization** | Executes **only once** at the beginning of the loop. | `int i = 0` |
| **Condition** | Checked **before** every iteration. If `true`, the loop continues. | `i < 10` |
| **Update** | Executes **after** the loop body runs, typically to increment/decrement the counter. | `i++` |

#### Example: Counting Up

```java
// Print numbers from 1 to 5
for (int i = 1; i <= 5; i++) {
    System.out.println("Iteration: " + i);
} 
// Output: 
// Iteration: 1
// ...
// Iteration: 5
```

#### Example: Counting Down

```java
// Print numbers from 10 down to 1
for (int i = 10; i >= 1; i--) {
    System.out.println(i);
}
// Output: 10, 9, 8, ..., 1
```

### 2\. The Enhanced `for` Loop (For-Each Loop)

Introduced in Java 5, the **Enhanced `for` Loop** (often called the **For-Each Loop**) is designed to simplify iteration over elements in **Arrays** and **Collections** (which we'll cover later).

  * **Key Feature:** It automatically manages the index/counter, making the code much cleaner and less error-prone when simply reading elements.

#### Syntax

```java
for (dataType elementVariable : arrayOrCollection) {
    // Code block that uses the elementVariable
}
```

| Component | Description |
| :--- | :--- |
| **`dataType elementVariable`** | The temporary variable that holds the current element during each iteration. Its type must match the elements in the array/collection. |
| **`arrayOrCollection`** | The array or collection you want to iterate through. |

#### Example: Iterating over an Array

```java
String[] names = {"Alice", "Bob", "Charlie"};

// Iterate over each String element in the names array
for (String name : names) { 
    System.out.println("Hello, " + name);
}

// Output:
// Hello, Alice
// Hello, Bob
// Hello, Charlie
```
