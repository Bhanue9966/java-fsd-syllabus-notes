### SQL Command Types and Explanation (Overview)

SQL commands are grouped into five main categories based on their function.

| Command Type | Full Form | Purpose | Key Commands |
| :--- | :--- | :--- | :--- |
| **DDL** | **Data Definition Language** | Defines, changes, and removes the **structure** (schema) of the database. | `CREATE`, `ALTER`, `DROP`, `TRUNCATE`, `RENAME` |
| **DML** | **Data Manipulation Language** | Handles access to and manipulation of the **data** within the tables. | `SELECT`, `INSERT`, `UPDATE`, `DELETE` |
| **DCL** | **Data Control Language** | Manages user **access and permissions** to the database. | `GRANT`, `REVOKE` |
| **TCL** | **Transaction Control Language** | Manages **transactions** (units of work) to ensure data integrity. | `COMMIT`, `ROLLBACK`, `SAVEPOINT` |
| **DQL** | **Data Query Language** | Strictly retrieves data from the database. (Often considered a subset of DML). | `SELECT` ||
